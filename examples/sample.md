# PagePress Sample Document

This is a sample document used to showcase PagePress PDF rendering.

## Feature Overview

PagePress supports the following:

- **Markdown to PDF**: full GFM syntax support
- **Syntax highlighting**: automatic language detection and highlighting
- **Multiple templates**: default, github, magazine

## Code Example

```typescript
import { render } from './pdf/renderer.ts';

const result = await render({
  input: 'document.md',
  output: 'document.pdf',
  template: 'default',
});

console.log('PDF generated:', result.pdfPath);
```
