CORE CONCEPT
PagePress 是什么？
PagePress 是一个 Node.js CLI 工具。输入 Markdown / HTML / URL，输出精美 PDF 或高清 PNG。内置 3 套 PDF 模板、8 种图片预设，2x Retina 输出，开箱即用。
pdf 命令 shot 命令 3 套模板 8 种预设

WORKFLOW
三种使用方式 — 从简单到进阶
L1 基础
命令行直出
一行命令，Markdown 直接转 PDF 或截图
pagepress pdf -i doc.md
L2 定制
HTML 模板
手写 HTML 布局，用 shot 命令渲染为高清 PNG
pagepress shot -i card.html
L3 自动化
AI + 批量生成
结合 Claude Skill，自然语言描述即可批量生成
配合 web-to-png Skill
MIT 开源
完全免费，可自由修改分发
2x Retina
默认高清输出，告别模糊截图
跨平台
macOS / Linux / Windows 全支持
BEST PRACTICES
六条使用建议
1
先写好 HTML 再截图
布局在浏览器调好，shot 命令一键输出
2
善用 --preset 参数
8 种预设自动设定宽高和缩放
3
--wait-until networkidle
加载 Web 字体时，等网络空闲再截图
4
PDF 选对模板
default / github / magazine 各有适用场景
5
图片设计避免 Web 元素
不用按钮、徽章、链接，保持海报感
6
结合 Skill 提效
安装 web-to-png Skill，AI 自动生成 HTML
npm i -g @liustack/pagepress
