Core Concept
PagePress 是什么？

PagePress 是一个 Node.js CLI 工具。输入 Markdown / HTML / URL，输出精美 PDF 或高清 PNG。内置 3 套 PDF 模板、8 种图片预设，2x Retina 输出，开箱即用。

三种使用方式 — 从简单到进阶
L1 基础

命令行直出

一行命令，Markdown 直接转 PDF 或截图

pagepress pdf -i doc.md
L2 定制

HTML 模板

手写 HTML 布局，shot 渲染高清 PNG

pagepress shot -i card.html
